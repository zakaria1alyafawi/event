# API Structure Summary\n\n## Overview\nFlask app factory in [`API/__init__.py`](API/__init__.py), registers blueprints under /api/v1:\n- [`login.py`](API/routes/login.py): POST /login (public)\n- tenant_management: POST /retrieve_users, /add_tenant, /delete_tenant, /update_tenant, /get_user (protected)\n- [`reset_password.py`](API/routes/reset_password.py), [`kill_session.py`](API/routes/kill_session.py)\n- HealthRoute imported but not registered\n\nAll routes extend [`BaseRoute`](API/routes/base.py): per-request DB session (appended to `g.sessions` for teardown), Blueprint auto-reg.\n\n## Authentication Flow\n- **Login**: email/pass → [`RetrieveUsers.validate_login`](Models/Users/RetrieveUsers.py) → [`AuthManager.generate_token`](API/middleware/auth.py) (HS256, 1h exp, user_id payload) → [`AddSessions`](Models/Sessions/AddSessions.py) (Token, Start/EndTime, Status=1)\n- **Protected**: Auth header Bearer TOKEN → [`AuthManager.authenticate_request`](API/middleware/auth.py): [`RetrieveSessions.get_by_token`](Models/Sessions/RetrieveSessions.py), check !expired & Status!=2 → UserID\n\n## Target Users & Use Cases\nMulti-tenant event mgmt:\n- **super_admin**: Full access (tenants/companies)\n- **event_admin**: Manage events/zones/attendance\n- **security**: Zone scans (enter/exit)\n- **exhibitor_staff**: Connections, booth mgmt\n- **visitor**: Attendance, networking\n\nTenants via `company_id` FK in Users.\n\n## Route Examples\n- **Login** (public): Returns token, UserType, names\n- **retrieve_users** (POST, auth): body {TenantID: int} → list users by company_id\n\n## Gaps\nNo routes for Events/Zones/etc. CRUD (models ready).\n\n```mermaid\ngraph TD\n    Client[Client] -->|POST /api/v1/login {email,pass}| Login[LoginRoute]\n    Login --> Validate[RetrieveUsers.validate_login]\n    Validate -->|ok| Token[AuthManager.gen_token]<br/>AddSession[AddSessions.add]<br/>\n    Client -->|Bearer TOKEN| Protected[retrieve_users etc.]\n    Protected --> Auth[AuthManager.authenticate_request]<br/>DBSession[RetrieveSessions.get_by_token & checks]\n    Auth -->|UserID| Handler[Business Logic]\n```\n
